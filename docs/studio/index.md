# Studio

Bloko Studio is the built-in admin interface for managing content. It provides a visual way to work with your collections, templates, and nodes.

## Launching Studio

From your project folder (where your `.env` file is located):

```bash
npx bloko studio
```

Studio will start at `http://localhost:4173`.

## Features

- **Collection Browser** - Navigate through collections and their node trees
- **Content Editor** - Edit node content in all configured languages
- **Template Manager** - Create and modify templates and their blocks
- **Relation Manager** - Create and manage relationships between nodes
- **Image Upload** - Upload and manage images with automatic variant generation
